Aula Git e GitHub

Versionamento de código na prática

O que é Git?

Git é um sistema de controle de versão usado para acompanhar alterações em arquivos de código ao longo do tempo.

Ele permite voltar versões, comparar mudanças e trabalhar em equipe sem bagunçar o projeto.

O que é GitHub?

GitHub é uma plataforma online que hospeda repositórios Git. É muito usada para compartilhar projetos, colaborar com outras pessoas e manter o código salvo na nuvem.

Comandos básicos do Git

  • git init – inicia um repositório
  • git status – mostra o estado dos arquivos
  • git add . – adiciona arquivos para commit
  • git commit -m – salva uma versão
  • git push – envia para o GitHub
